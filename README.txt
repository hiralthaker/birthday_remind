INTRODUCTION
------------

The Birthday Reminder module allow to add birthday of people and give reminder on site.

By default, the module creates a block of person who has birthday on that day.

This module creates a new content type named 'Birthday' to add/edit/delete birthday of people

CONFIGURATION
-------------

To configure Birthday Reminder, go to Administration > Configuration > Development > Birthday Reminder Settings.

FEATURES
--------

Birthday Reminder Module gives below facility:

1. Allow to add people's birthday and related information.
2. Show people's birthday on site
3. Send mail to all people for whishing them on their birthday.
